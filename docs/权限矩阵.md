# 权限矩阵

## 参与方与权责

| 参与方 | 主体责任 | 平台权限 |
|---|---|---|
| 预制构件工厂 | 生产质量、出厂检验结论 | 录入构件、出厂登记、查看本工厂构件 |
| 运输单位 | 运输过程构件完好、路线合规 | 上传轨迹与温湿度、查看本运输单位的告警 |
| 施工总承包 | 进场验收、吊装、节点连接、成品保护 | 进场、吊装、节点、保护登记 |
| 监理单位 | 隐蔽验收、质量评定 | 隐蔽工程验收、查看相关构件 |
| 建设单位 | 多参与方协调、档案归档 | 项目总览、生成/下载/报送档案 |
| 质量监督机构 | 监督抽检、行政处罚 | 查看告警、签收档案 |

## 离线同步

- 工地扫码终端通过 `transport/telemetry` 与 `/api/sync/batch` 上传
- 离线事件先入 IndexedDB，恢复网络后由前端自动调用 `POST /api/sync/batch`
- 后端基于请求方角色与参与方 ID 校验事件合法性
